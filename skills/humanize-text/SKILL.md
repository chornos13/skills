Model prose carries **tells** — habits that mark text as machine-written. Hunt every tell below and rewrite each hit in the **spoken** register: how the author would say it out loud, to a person.

Change how it sounds, never what it claims. Facts, numbers, names, and the author's position survive intact.

## Process

1. Mark every tell present in the text.
2. Rewrite each marked passage into the spoken register, using the fix beside the tell.
3. Read it back the way you'd say it. Any sentence you wouldn't actually say goes back to step 2.

Done when every tell below has been checked against the full text and each hit is cleared.

## The tells

Each tell carries its fix.

**Symbols**
- Em dash as the default connector → period, comma, parentheses, or "and"
- Curly quotes and apostrophes → straight ones; a person typing produces straight
- Emoji decorating headers or bullets → cut
- Bold sprayed on phrases mid-paragraph → cut, or restructure so the sentence carries the emphasis
- Bullets where prose would do → prose

**Phrasing**
- "It's not just X, it's Y" and "not only… but also" → state the claim once
- Rule-of-three lists ("fast, reliable, and scalable") → name the one that matters
- Elevated synonyms (delve, leverage, robust, seamless, tapestry, realm, landscape, testament, underscore, pivotal, foster, harness, unlock, myriad) → the plain word you'd say: use, dig into, strong, big
- Connectors nobody speaks (furthermore, moreover, additionally) → and, also, or nothing
- Hedge preamble ("It's worth noting that", "It's important to remember") → ope
- Opener throat-clearing ("In today's fast-paced world", "Let's dive in") → start at the first real sentence
- Closing bow ("In conclusion", "Overall", "I hope this helps!") → end on the l
- A final sentence restating the paragraph above it → delete

**Rhythm** — the loudest tell
- Every sentence the same length → go **bursty**: a nine-word sentence beside agment
- Every paragraph the same size → let one run long and one land in a single line
- Every list item the same shape → break the parallelism
- No contractions → it's, don't, you'd
- No sentence opening on And, But, or So → open some
